# 20 Git Commands and Their Functions

1. **git init:**
   - Initializes a new Git repository.

2. **git clone:**
   - Clones a repository into a new directory.

3. **git add:**
   - Adds changes in the working directory to the staging area.

4. **git commit:**
   - Records changes to the repository with a message.

5. **git status:**
   - Shows the status of changes as untracked, modified, or staged.

6. **git push:**
   - Pushes changes to a remote repository.

7. **git pull:**
   - Fetches changes from a remote repository and merges them.

8. **git branch:**
   - Lists existing branches or creates a new one.

9. **git merge:**
   - Merges two branches together.

10. **git log:**
    - Displays the commit history.

11. **git diff:**
    - Shows changes between commits, commit and working tree, etc.

12. **git checkout:**
    - Switches branches or restores working tree files.

13. **git reset:**
    - Resets current HEAD to the specified state.

14. **git remote:**
    - Manages remote repositories.

15. **git fetch:**
    - Downloads objects and refs from another repository.

16. **git rebase:**
    - Combines a sequence of commits into a new base commit.

17. **git tag:**
    - Creates, lists, deletes, or verifies a tag object signed with GPG.

18. **git log:**
    - Displays the commit history.

19. **git show:**
    - Shows various types of objects.

20. **git stash:**
    - Stashes changes in a dirty working directory away.
